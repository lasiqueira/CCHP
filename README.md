# CCHP
"Clean" Code Horrible Performance inspired benchmarks but in Kotlin
## Setup
You are required to have JDK 17 installed;

## Build
`./gradlew clean build`

## Running benchmarks
`./gradlew clean jmh`
## Sample
Specs:

CPU: AMD Zen 4 R7 7800X3D

Memory: 32GB DDR5 6000Mhz

GPU: NVIDIA RTX 4090
